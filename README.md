# System Parameters Manager (spmd)

## Описание

System Parameters Manager - это демон, следящий за параметрами системы. Параметры, в зависимости от значений, сохраняются и выводятся индикации о выходе за установленные пределы.

## Технические требования к реализации

ОЗУ:
- Мониторинг общего объема ОЗУ.
- Мониторинг объема используемой ОЗУ.
- Мониторинг объема свободной ОЗУ.
SWAP:
- Мониторинг общего объема файла подкачки (swap).
- Мониторинг объема используемого файла подкачки (swap).
DIR:
- Мониторинг общего объема места директории.
- Мониторинг занятого места директорией.
- Мониторинг свободного места директорией.
Логирование:
- Сохранение времени выхода/входа за/в установленный предел какого-либо из параметров.
Вывод:
- Вывод текущих параметров по запросу.
- Вывод имеющихся аварий (выход параметра за предел) по запросу.
Очистка:
- Очистка имеющихся аварий по запросу.
Всплывающее окно:
- При обнаружении выхода за установленный предел вывести всплывающее окно с сообщением.
Другое:
- Добавить гистерезис для предотвращения ложного срабатывания/сброса аварий.
- Система сборки должна основываться на make/cmake.

## Список параметров

- Объем используемой ОЗУ.
- Объем свободного ОЗУ.
- Объем занятого файла подкачки (swap).
- Объем занятого места директории.
